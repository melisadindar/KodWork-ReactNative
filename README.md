# KodWork

React Native ile geliştirilmiş bir iş ilanı uygulaması.

## Özellikler
- The Muse API'dan iş ilanlarını listeler
- İlan detaylarını görüntüler
- İlanları favorilere ekler ve çıkarır
- Redux ile state yönetimi

## Kullanılan Teknolojiler
- React Native
- TypeScript
- Redux Toolkit
- React Navigation (Drawer + Stack)
- React Native Vector Icons

## Kurulum
npm install
cd ios && pod install && cd ..
npx react-native run-ios
